About Cats Curios

Cats Curios is a fictional blog built to practice modern Django development. Each post is a small “adventure” from a curious cat, with moods, images, and community comments.

Technologies

  • Django
  • Bootstrap 5
  • Cloudinary (media storage)
  • WhiteNoise (static files)

Learning goals

  • Authentication (login/logout/register)
  • CRUD patterns with class-based views
  • Relational data (posts + comments + profiles)
  • Deployment-ready configuration

Disclaimer: This is a student/portfolio project and the content is fictional.
